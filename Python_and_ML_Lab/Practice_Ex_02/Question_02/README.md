# Question 02

Write a python code to add two numbers using input() function.

### Sample Testcases

#### Input 1

```	
5
3
```

#### Output 1

```
The sum of 5 and 3 is 8.0
```

#### Input 2

```
3.5
2.8
```

#### Output 2

```
The sum of 3.5 and 2.8 is 6.3
```

#### Input 3

```
99
72.6
```

#### Output 3

```
The sum of 99 and 72.6 is 171.6
```
